🔐 WebAuthn/FIDO2 Passwordless Authentication Demo

Demo đăng nhập không mật khẩu sử dụng WebAuthn/FIDO2 với Spring Boot

📝 Đăng ký thiết bị mới

🔑 Đăng nhập bằng WebAuthn

📋 Thông tin API

Base URL: http://localhost:8080

Registration Endpoint: POST /api/webauthn/registration/options

Authentication Endpoint: POST /api/webauthn/assertion/options

Server: Spring Boot + Yubico WebAuthn
